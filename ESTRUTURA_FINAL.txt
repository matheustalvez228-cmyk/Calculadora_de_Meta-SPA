🚀 COMO USAR ESTE PROJETO
═══════════════════════════════════════════════════════════════════════════════

1️⃣ CLONAR REPOSITÓRIO
   git clone https://github.com/seu-usuario/sistema-meta-concilig.git
   cd sistema-meta-patched

2️⃣ INSTALAR DEPENDÊNCIAS
   cd backend
   npm install

3️⃣ INICIALIZAR BANCO DE DADOS
   node init_db.js

4️⃣ INICIAR SERVIDOR
   npm start
   Aguarde: "Server running on port 3000"

5️⃣ ABRIR NO NAVEGADOR
   http://localhost:3000

6️⃣ FAZER LOGIN
   Matrícula: 10001 (ou qualquer outro user)
   Senha: 123456
